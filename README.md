## 환경설정
* virtualenv를 이용한 환경설정
  $ sudo apt install python3-virtualenv
  $ cd ~/workspace
  $ virtualenv new_project
  $ cd new_project
  $ source bin/activate
  (new_project) $ pip install numpy, pandas, matplotlib, scikit-learn, keras, jupyter
  (new_project) $ jupyter-notebook
